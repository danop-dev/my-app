## What is this?

`my-app` is a full-stack **Turborepo monorepo** with a shared UI library, NestJS backend API, Next.js frontend, and this documentation site.

## Apps

| App | Port | Description |
|-----|------|-------------|
| `apps/web` | `3000` | Next.js 16 frontend |
| `apps/api` | `8000` | NestJS 11 REST API |
| `apps/docs` | `3030` | This documentation site |

## Packages

| Package | Description |
|---------|-------------|
| `packages/ui` | Shared React component library |
| `packages/backend-config` | Shared NestJS guards, pipes, filters, interceptors, decorators |
| `packages/eslint-config` | Shared ESLint configs for frontend and backend |
| `packages/typescript-config` | Shared TypeScript configs |

## Quick Start

```bash
# Install dependencies
pnpm install

# Run all apps
pnpm dev

# Run a single app
pnpm dev --filter=web
pnpm dev --filter=@workspace/api
pnpm dev --filter=docs
```
